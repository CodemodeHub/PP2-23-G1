Part I
1) Write a regular expression that matches any string that contains the word "hello".

2) Write a regular expression that matches any string that starts with the word "hello".

3) Write a regular expression that matches any string that ends with the word "world".

4) Write a regular expression that matches any string that contains a digit (0-9).

5) Write a regular expression that matches any string that contains only letters 
(no numbers or special characters).

6) Write a regular expression that matches any string that contains only uppercase letters.

7) Write a regular expression that matches any string that contains only lowercase letters.

8) Write a regular expression that matches any string that contains a combination of letters 
and numbers, but no special characters.

9) Write a regular expression that matches any string that contains a word that starts with 
the letter "a" and ends with the letter "z".

10) Write a regular expression that matches any string that contains a URL 
(e.g. "https://www.example.com").

Part II

1) Write a Python program that matches a string that contains the word "Python".

2) Write a Python program that matches a string that starts with "The" 
and ends with "Kazakhstan".

3) Write a Python program to match a string that contains only upper and lowercase letters, 
numbers, and underscores.

4) Write a Python program to match a string that contains a date in the format "dd/mm/yyyy".

5) Write a Python program to match a string that contains a valid email address.
